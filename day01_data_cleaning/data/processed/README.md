# 📂 Processed Data

## Descripción

Esta carpeta contiene el dataset limpio y preparado para análisis posterior.

## Archivo

- `Dataset Limpio.xlsx`: versión limpia del dataset original de ventas retail de e-cigarettes.

## Uso

Este archivo será utilizado en las siguientes etapas del proyecto para:

- análisis en Excel;
- tablas dinámicas;
- KPIs;
- visualizaciones;
- construcción del dashboard final.

## Nota

El dataset procesado resulta de la limpieza del archivo original ubicado en la carpeta `raw`.
