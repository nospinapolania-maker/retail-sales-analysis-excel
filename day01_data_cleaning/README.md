# 🧹 Día 1 · Limpieza de Dataset Retail en Excel

## ✅ Estado

Completado.

## 📝 Descripción

Este mini proyecto corresponde al **Día 1 de la Semana 1** del bootcamp autodirigido **_Data Analyst: 0 to Hero-16 Weeks-_** el cual se enfoca en análisis de ventas.

El objetivo fue limpiar un dataset sucio de ventas retail en Excel para dejarlo preparado para análisis posteriores, tablas dinámicas y construcción de dashboard.

## 🎯 Objetivo técnico

Aplicar limpieza básica de datos en Excel para mejorar la calidad del dataset y prepararlo para análisis.

## 🔧 Acciones realizadas

- Revisión inicial del dataset.
- Identificación de datos inconsistentes.
- Corrección de formatos.
- Organización de columnas.
- Preparación del archivo limpio para análisis posterior.

## 🛠️ Herramientas utilizadas

- Microsoft Excel(Power Query)
