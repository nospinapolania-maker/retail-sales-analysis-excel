# 🧹 Día 1 · Limpieza de Dataset Retail en Excel

Completado.

## 📝 Descripción

Este mini proyecto corresponde al **Día 1 de la Semana 1** del bootcamp autodirigido **Data Analyst: 0 to Hero - 16 Weeks**, el cual se enfoca en análisis de ventas y construcción progresiva de un portafolio profesional.

El objetivo fue limpiar un dataset sucio de ventas retail en Excel para dejarlo preparado para análisis posteriores, tablas dinámicas, KPIs, visualizaciones y construcción de dashboard.

Este ejercicio hace parte de la primera etapa del proyecto de Excel, donde el foco principal es preparar correctamente los datos antes de analizarlos.

---

## 📊 Dataset utilizado

Para este mini proyecto se trabajó con un dataset panel de ventas minoristas de **e-cigarettes** en Estados Unidos.

El dataset registra ventas retail en los **50 estados de EE. UU. y Washington D.C.**, desde **febrero de 2020 hasta septiembre de 2025**. Cada fila representa una combinación de:

- mes;
- estado;
- marca;
- sabor;
- tipo de producto.

El archivo original contiene **116.526 registros** y permite analizar ventas, marcas, sabores, tipos de producto, volumen de nicotina y variables regulatorias asociadas al mercado de e-cigarettes.

---

## 🔎 Características del dataset

Este dataset combina información comercial y regulatoria, incluyendo:

- ventas mensuales por estado;
- marca del producto;
- sabor;
- tipo de producto;
- estado de prohibición de sabores por estado;
- fecha efectiva de regulación;
- estado de autorización FDA PMTA por marca;
- tasas estatales de impuestos especiales;
- volumen de nicotina vendido.

En esta etapa, el objetivo no fue realizar un análisis profundo del mercado, sino preparar el dataset para futuras fases del proyecto.

---

## 🎯 Objetivo técnico

Aplicar limpieza básica de datos en Excel para mejorar la calidad del dataset y prepararlo para análisis.

El propósito fue dejar una versión más organizada y confiable del archivo, lista para ser usada en:

- tablas dinámicas;
- análisis de ventas;
- KPIs;
- gráficos;
- dashboard final de la Semana 1.
